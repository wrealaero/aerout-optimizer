## what it does

- **dashboard** - live health score that shows you exactly whats dragging it down, junk, memory, disk, security, with the real math behind it
- **ai optimizer** - reads your actual stats and gives you fixes for whats actually going on, shows the command before it runs
- **analysis** - live cpu, ram, gpu, disk and network stats + a speed test
- **cleaner** - clears caches, logs, temp files, game junk and duplicate downloads, shows the size before you clean
- **tweaks** - a big list of reversible system tweaks you flip on and off, everything reverts clean
- **security** - quick or deep scan for malware, miners, spyware, sus launch agents and hijacked browser extensions
- **license** - your tier, time left and whether your mac is bound
- **settings / about** - scans, notifications, menu bar, auto updates and version stuff

## getting it

grab the latest zip from [releases](../../releases), open it in your downloads, then drag AeroutOptimizer.app into your applications folder and open it.

first time you open it macos might say its from an unidentified developer. thats normal. just right click the app, hit open, then open again on the popup.

or if you want, open terminal and paste this then hit enter:
```bash
xattr -cr /Applications/AeroutOptimizer.app
```

after that it opens fine every time. auto updates are built in so youre always on the latest.
